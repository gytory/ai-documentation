# Корпоративный помощник (Сайт-помощник для ведения корпоративной документации)

## О проекте

Веб-приложение для централизованного хранения корпоративной документации с возможностью быстрого поиска информации через нейросеть (RAG-система). Сотрудники могут задавать вопросы на естественном языке и получать ответы с указанием источников.

## Функциональные возможности

- **Авторизация и роли**: пользователь, модератор, администратор
- **Управление документами**: загрузка, просмотр, редактирование, удаление
- **Поиск с ИИ**: задавайте вопросы по конкретному документу или всем сразу
- **Чат с нейросетью**: получение ответов с привязкой к источникам
- **Администрирование**: управление пользователями и их ролями

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Фронтенд | React 18, React Router, Axios |

## Быстрый старт

### Требования

- Python 3.10+
- Node.js 18+
- PostgreSQL

### Начало работы

1. **Клонировать репозиторий**
```bash
git clone https://github.com/gytory/ai-documentation
```

2. **Настройка бэкенда**
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

3. **Настройка базы данных**
```sql
CREATE DATABASE corporate_assistant;
-- настроить подключение в .env или
--заменить строку DATABASE_URL в файле database.py на свои данные
```

4. **Запуск Ollama**
```bash
ollama pull granite4:350m-h
ollama serve
```

5. **Запуск бэкенда**
```bash
uvicorn main:app --reload --port 8000
```

6. **Настройка фронтенда**
```bash
cd frontend
npm install
npm start
```

## API Эндпоинты (основные)

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/auth/register` | Регистрация |
| POST | `/auth/login` | Вход |
| GET | `/documents` | Список документов |
| POST | `/documents` | Загрузка документа |
| POST | `/rag/ask` | Запрос к нейросети |
| GET | `/users` | Список пользователей (админ) |

## Роли и права

- **Пользователь (user)**: просмотр документов, вопросы к ИИ
- **Модератор (moderator)**: всё + управление документами
- **Администратор (admin)**: всё + управление пользователями

## Демонстрация работы системы

- Главная страница со списком документов
- Чат с нейросетью (история, источники ответов)

- Административные панели
